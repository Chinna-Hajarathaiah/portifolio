Health Monitoring Tool

Tech Stack: Spring Boot, MySQL, Actuator, Quartz Scheduler, JPA, Chart.js, Docker, Postman, HTML, CSS and JS
This project is designed to monitor and calculate the health conditions of applications in real-time. It collects and analyzes key metrics such as uptime, response time, memory usage, and network bandwidth using Spring Boot Actuator and Quartz Scheduler. The system integrates the metric data into an interactive dashboard using Chart.js, providing real-time visual insights into application performance. It also supports dynamic monitoring of multiple applications, making it an essential tool for performance tracking and issue resolution.
